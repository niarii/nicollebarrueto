Proyecto 1: Resistencia

Descripción: Una resistencia es un componente eléctrico que limita la cantidad de corriente que puede fluir a través de un circuito. Es crucial en muchos circuitos electrónicos para proteger componentes sensibles.

Materiales:
- Arduino Uno
- Protoboard
- Resistencia (220Ω)
- LED
- Cables de conexión

Pasos:
1. Conexión de la Resistencia y el LED:
- Conecta la pata larga del LED (ánodo) al pin digital 13 del Arduino a través de una resistencia de 220Ω.
- Conecta la pata corta del LED (cátodo) a GND (tierra) del Arduino.
2. Código Arduino IDE:
```cpp void setup() { pinMode(13, OUTPUT); // Configura el pin 13 como salida } void loop() { digitalWrite(13, HIGH); // Enciende el LED delay(1000); // Espera 1 segundo digitalWrite(13, LOW); // Apaga el LED delay(1000); // Espera 1 segundo } ```

3. Sube el código al Arduino:
- Conecta el Arduino a tu computadora.
- Abre el Arduino IDE, pega el código anterior y súbelo al Arduino.

Proyecto 2: Sensor de Ultrasonido con Botón

Descripción:
Este proyecto utiliza un sensor de ultrasonido para medir la distancia y un botón para activar el sensor. El resultado se mostrará en el monitor serie del Arduino.

Materiales:
- Arduino Uno
- Sensor de Ultrasonido HC-SR04
- Botón
- Protoboard
- Cables de conexión
- Resistencia (10kΩ para el botón)

Pasos:
1. Conexión del Sensor de Ultrasonido:
- VCC del sensor al pin 5V del Arduino.
- GND del sensor a GND del Arduino.
- Trig del sensor al pin digital 9 del Arduino.
- Echo del sensor al pin digital 10 del Arduino.
2. Conexión del Botón:
- Un pin del botón al pin digital 2 del Arduino.
- El otro pin del botón a GND.
- Coloca una resistencia de 10kΩ entre el pin del botón que está conectado al pin 2 del Arduino y el 5V.
3. Código Arduino IDE:
```cpp const int trigPin = 9; const int echoPin = 10; const int buttonPin = 2; long duration; int distance; void setup() { pinMode(trigPin, OUTPUT); pinMode(echoPin, INPUT); pinMode(buttonPin, INPUT_PULLUP); Serial.begin(9600); } void loop() { if (digitalRead(buttonPin) == LOW) { digitalWrite(trigPin, LOW); delayMicroseconds(2); digitalWrite(trigPin, HIGH); delayMicroseconds(10); digitalWrite(trigPin, LOW); duration = pulseIn(echoPin, HIGH); distance = duration * 0.034 / 2; Serial.print("Distance: "); Serial.print(distance); Serial.println(" cm"); delay(1000); } } ```

4. Sube el código al Arduino:
- Conecta el Arduino a tu computadora.
- Abre el Arduino IDE, pega el código anterior y súbelo al Arduino.
- Abre el monitor serie para ver la distancia medida cuando presionas el botón.

Proyecto 3: Semáforo en Tinkercad

Descripción:
Este proyecto simula un semáforo usando LEDs. Los LEDs se encenderán en secuencia para simular las luces de un semáforo: rojo, amarillo y verde.

Materiales:
- Arduino Uno
- LEDs (rojo, amarillo y verde)
- Resistencias (220Ω)
- Protoboard
- Cables de conexión

Pasos:
1. Conexión de los LEDs:
- Conecta la pata larga de cada LED a los pines digitales del Arduino: rojo al pin 13, amarillo al pin 12, y verde al pin 11.
- Conecta la pata corta de cada LED a GND a través de una resistencia de 220Ω.
2. Código Arduino IDE:
```cpp const int redLED = 13; const int yellowLED = 12; const int greenLED = 11; void setup() { pinMode(redLED, OUTPUT); pinMode(yellowLED, OUTPUT); pinMode(greenLED, OUTPUT); } void loop() { digitalWrite(redLED, HIGH); delay(5000); digitalWrite(redLED, LOW); digitalWrite(yellowLED, HIGH); delay(2000); digitalWrite(yellowLED, LOW); digitalWrite(greenLED, HIGH); delay(5000); digitalWrite(greenLED, LOW); } ```

3. Sube el código al Arduino:
- Conecta el Arduino a tu computadora.
- Abre el Arduino IDE, pega el código anterior y súbelo al Arduino.

Proyecto 4: Sensor de Ultrasonido con Servomotor

Descripción:
Este proyecto utiliza un sensor de ultrasonido para detectar objetos y mover un servomotor para abrir una puerta cuando la distancia medida es menor a un umbral determinado.

Materiales:
- Arduino Uno
- Sensor de Ultrasonido HC-SR04
- Servomotor
- Protoboard
- Cables de conexión

Pasos:
1. Conexión del Sensor de Ultrasonido:
- VCC del sensor al pin 5V del Arduino.
- GND del sensor a GND del Arduino.
- Trig del sensor al pin digital 9 del Arduino.
- Echo del sensor al pin digital 10 del Arduino.
2. Conexión del Servomotor:
- VCC del servomotor al pin 5V del Arduino.
- GND del servomotor a GND del Arduino.
- Pin de control del servomotor al pin digital 6 del Arduino.
3. Código Arduino IDE:
```cpp include const int trigPin = 9; const int echoPin = 10; const int servoPin = 6; long duration; int distance; Servo myServo; void setup() { pinMode(trigPin, OUTPUT); pinMode(echoPin, INPUT); myServo.attach(servoPin); myServo.write(0); // Cierra la puerta inicialmente Serial.begin(9600); } void loop() { digitalWrite(trigPin, LOW); delayMicroseconds(2); digitalWrite(trigPin, HIGH); delayMicroseconds(10); digitalWrite(trigPin, LOW); duration = pulseIn(echoPin, HIGH); distance = duration * 0.034 / 2; Serial.print("Distance: "); Serial.print(distance); Serial.println(" cm"); if (distance < 10) { myServo.write(90); // Abre la puerta } else { myServo.write(0); // Cierra la puerta } delay(1000); } ```

4. Sube el código al Arduino:
- Conecta el Arduino a tu computadora.
- Abre el Arduino IDE, pega el código anterior y súbelo al Arduino.
- Observa cómo el servomotor se mueve para abrir y cerrar la puerta basada en la distancia medida por el sensor de ultrasonido.
